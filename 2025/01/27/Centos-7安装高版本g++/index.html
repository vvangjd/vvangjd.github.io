注意事项

在使用VSCode作为代码编辑器时，可能在#include <iostream>等头文件下方出现红色波浪线，
这是因为编辑器找不到g++的路径导致的。

解决方法：
Mac: command + shift + p -> C/C++ configurations(UI) -> compiler path处填入
Win: Ctrl + shift + p -> C/C++ configurations(UI) -> compiler path处填入
